Dynamic Many2One Field Options Label Customizer (Odoo17)

The Dynamic Many2One Field Label Customizer allows you to personalize and enhance the user experience by modifying the default labels of Many2One field dropdowns in Odoo. With this app, you can easily configure custom labels like Create, Create and edit..., Search More..., Start typing..., and No records for any model(s) Many2One fields. This flexibility enables you to tailor the interface to specific business needs, making it more intuitive and user-friendly.

The app is configuration-driven, allowing you to define custom labels for any model and activate them as needed. For example, you can apply custom labels to the "Employee" model's department_id field, providing a more meaningful user experience. The configuration comes with an active toggle, so you have full control over when and where these label changes are applied.

Key Features

  • Customizable Labels for Many2One Fields:Configure the labels for commonly used options such as Create, Create and edit..., Search More..., Start typing..., and No records.
  • Select the Model:Choose the model (e.g., Employee) where you want to customize the labels for Many2One fields.
  • Add Many2One Fields:In the one2many section, select the Many2One fields (e.g., department_id) you want to apply the custom labels to.
  • Define Custom Labels:Specify the labels for Create, Create and edit..., Search More..., Start typing..., and No records
  • Activate Customization:Enable the Active checkbox to apply the configuration.
  • Save and Apply:Once saved, the customized labels will be reflected immediately in the specified model’s Many2One fields.
