Dynamic Multi-Field Name Search and Get for Many2One Fields (Odoo17)

Enhance the functionality of Many2One fields in Odoo with dynamic multi-field Search and Get capabilities. This app allows users to search and get records within Many2One fields using multiple fields. With a user-friendly configuration interface, you can easily specify the fields to be included in the search criteria, making it more efficient and flexible for finding records as well as displaying the records based on multiple fields instead of only name field.

Key Features

  • Dynamic Multi-Field Search:Search within Many2One fields using multiple fields.
  • Display Multi-Field in Many2one:Display Multiple Fields in Many2one instead of default name field.
  • Easy Configuration:Specify the fields for each model through a simple configuration interface.
  • Enhanced Search Accuracy:Improve search results by including additional relevant fields.
  • Enhanced Many2one Field Display:Improve display results by including additional relevant fields.
  • Seamless Integration:Works with all Many2One fields across different Odoo interfaces.
  • Active/Inactive Toggle:Easily activate or deactivate the name search and name get configuration for any model.

Configuration

  • Navigate to Name Search and Get Config:Go to the Odoo menu and click on the "Name Search and Get Config" menu.
  • Create New Configuration:Click on "Create" to add a new configuration record.
  • Choose Model:Choose the model (e.g., res.partner) for which you want to enable multi-field name search and get.
  • Choose Fields:Add the fields (e.g., email, phone) on which you want to perform the search in the Many2One field and display that fields on many2one.
  • Active/Inactive Toggle:Activate or deactivate the search and get configuration for any model.
  • Update models.py File Once:No technical and programming knowledge is required. You just need to copy and paste the code into the relevant models.py file once (Please check video for details).
