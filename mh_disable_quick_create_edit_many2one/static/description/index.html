Disable or Hide Quick Create and Edit Option for Many2One Fields Dynamically (Odoo16)

Empower your Odoo experience with the ability to dynamically control the quick create and edit options for Many2One fields based on configuration (without using any widget for specific fields). This app enhances usability by allowing administrators to manage which fields and models should disable these features, optimizing user workflows and data integrity.

Key Features

  • Dynamic Configuration:Customize which models and fields should disable quick create and edit functionalities. For instance, if you need to hide the create and edit options for a specific Many2One field within a particular model, this app is perfect for you.
  • Flexible Control:Administrators can easily toggle these settings based on specific business requirements.
  • Enhanced Usability:Improves user experience by reducing clutter and ensuring data consistency.
  • Easy Setup:Configure settings directly from the Odoo interface without requiring code modifications.
  • Scalable Solution:Adaptable for different modules and fields, ensuring flexibility across various business processes.

Configuration

  • Menu:Go to Disable Quick Create menu

  • Create Configuration:Create new record, Select model and fields in which you want to disable quick create and edit option.

  • Testing Time:Navigate to the selected model and inspect the many2one field. The options to Create and Edit will be hidden.
