Dynamic Many2One Field Option Configuration Based (Odoo16)

Introducing the Dynamic Many2One Field Configuration app for Odoo16 This versatile module allows you to dynamically customize the behavior of many2one fields across your Odoo instance. With this app, you can control the number of items displayed in dropdowns, manage search result limits, and simplify your interface by removing unnecessary options. Perfect for streamlining your workflow and improving user experience. You can even configure these settings individually for each many2one field within a model, providing unparalleled flexibility.

Key Features

  • Set Dropdown Limit:Configure the number of items displayed in the search dropdown. Default is set to 7 items.
  • Set Search More Limit:Adjust the limit for the Search More functionality. Default is set to 320 items.
  • Remove Quick Create Option:Disable the quick create option in many2one fields to ensure data consistency.
  • Remove Create and Edit Option:Remove the Create and Edit option to prevent unintentional record creation.
  • Remove Open Option:Eliminate the Open option to streamline the user interface.
  • Field-Specific Configuration:Set different configurations for each many2one field within a model. For example, in the Sale Order model, customize settings for each many2one field independently.

Configuration

  • Installation:Install the app from the Odoo App Store.
  • Settings:Navigate to the app's configuration page to set the desired limits and options.
  • Field-Specific Configuration:Customize settings for individual many2one fields within a specific model.
