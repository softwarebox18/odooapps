Star Rating Dynamic Widget for Integer Field (Odoo16)

The Star Rating Widget for Integer Fields is a dynamic and customizable tool designed for Odoo users to add interactive star ratings to integer fields in their forms. With options to configure the number of stars, filled/unfilled colors, spacing, and star size, this widget allows users to provide visually appealing and easy-to-use rating inputs for various applications such as employee performance, product reviews, customer satisfaction, and more.

The widget integrates seamlessly into Odoo and enhances the user experience by allowing for custom rating systems.

Configuration

How to Use the Widget in an Integer Field

  • Assign the widget to any integer field in a model by adding the following to the form view
  • Add the star_rating widget to the integer field using the XML view definition. widget="star_rating"
  • Configure the options {'maxStars': 10, 'filledColor': 'golden', 'unfilledColor': 'grey', 'starSpacing': 3, 'starWidth': 30}

Explanation of Options

  • maxStars:Number of stars for the rating (default: 5)
  • filledColor:Color of the filled stars (default: golden)
  • unfilledColor:Color of the unfilled stars (default: grey)
  • starSpacing:Space between stars in pixels (default: 4)
  • starWidth:Width of each star in pixels (default: 20)
