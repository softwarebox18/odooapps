Remove POS Order Lines

This module enhances the Point of Sale (POS) interface by providing users with the flexibility to manage their order lines efficiently. Users can remove individual order lines by clicking the dedicated remove button next to each item. Additionally, the module introduces a 'Clear All Items' button, allowing users to remove all order lines simultaneously with a single click.

Configuration

  • Just install the module mh_remove_pos_orderline
